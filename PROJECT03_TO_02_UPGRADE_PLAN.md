# 프로젝트03 → 프로젝트02 역이식 계획

## 📋 반영 가능한 개선사항

### ✅ 1. 성능 최적화 (즉시 적용 가능)

**Lazy Loading 이미지**
```html
<!-- 현재 프로젝트02 -->
<img src="image.jpg">

<!-- 프로젝트03 방식 적용 후 -->
<img src="image.jpg" loading="lazy" class="lazy-img">
```

**이미지 최적화 CSS/JS**
```css
img[loading="lazy"] {
    opacity: 0;
    transition: opacity 0.3s ease;
}
img[loading="lazy"].loaded {
    opacity: 1;
}
```
